# Mac Start Guide

This project is an Electron app. The Windows `.bat` file cannot run on macOS.

## First Time Setup

1. Open https://nodejs.org/en/download in Safari or Chrome.
2. Download and install the LTS version for macOS.
3. Come back to this folder.
4. Double-click `Start Desktop Cat Reminder.command`.

The first launch may take a little while because it installs the app dependencies.

## If macOS Blocks The File

If macOS says the command file cannot be opened, open Terminal and run:

```bash
cd /Users/zhengziqi/Documents/GitHub/Desktop-Cat-Reminder
chmod +x "Start Desktop Cat Reminder.command"
```

Then double-click `Start Desktop Cat Reminder.command` again.
